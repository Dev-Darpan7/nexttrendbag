#  NextTrendBag

NextTrendBag is a responsive e-commerce website developed using **PHP, MySQL, JavaScript, HTML, and CSS**. The project provides a complete online shopping experience, allowing users to browse products, manage their cart and wishlist, place orders, and manage their accounts through a clean and user-friendly interface.

---

## 🚀 Features

### 👤 User Authentication

* User registration
* Secure login and logout
* Forgot password functionality
* Change password
* Profile management

### 🛒 Shopping

* Browse products
* Product details page
* Add products to cart
* Update or remove cart items
* Wishlist functionality
* Checkout system
* Place orders
* Order success page

### 📦 Order Management

* View order history
* View individual order details
* Cancel orders

### 📄 Additional Pages

* Home page
* Shop page
* About page
* Contact page

### 🛠️ Admin Panel

* Product management
* Order management
* User management

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Dev-Darpan7/nexttrendbag.git
```

### 2. Move the project

Copy the project folder into your XAMPP `htdocs` directory.

Example:

```text
C:\xampp\htdocs\nexttrendbag
```

### 3. Import the database

* Start **Apache** and **MySQL** in XAMPP.
* Open **phpMyAdmin**.
* Create a database (for example, `nexttrendbag`).
* Import the `nexttrendbag.sql` file.

### 4. Configure the database

Open `db.php` and update the database credentials if needed.

### 5. Run the project

Visit:

```text
http://localhost/nexttrendbag
```
